📝 Cara Menggunakan Multi Grup:

  1. Tambah Grup: Isi nama grup dan klik "Tambah Grup"
  2. Pilih Grup: Klik tab grup yang ingin diedit
  3. Input Tim: Tulis nama tim di grup tersebut
  4. Update Tim: Klik "Update Tim Grup" untuk menyimpan
  5. Generate: Klik "Generate Semua Liga" atau per grup
  6. Mainkan: Input skor dan submit seperti biasa

🎯 Fitur:

  • ✅ Multi grup dalam satu liga
  • ✅ Masing-masing grup punya jadwal sendiri
  • ✅ Auto-save ke localStorage
  • ✅ Undo hingga 50 langkah
  • ✅ Export klasemen
  • ✅ Responsive untuk semua device
